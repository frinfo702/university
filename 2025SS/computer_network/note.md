# week4~6の資料

シーケンスナンバー(Sequence Number: SN)
送信データの先頭から順に付けたデータ番号（Byte単位）
先頭の番号はOSがつける（安全のため予測できないようにランダム）
受信側ではこれをもとに並べ直す。
消失したセグメントも検出できる
上り（発呼側→着呼側）と下り（着呼側→発呼側）の２つ
独立に存在する
